## 应用层

应用层主要提供用户交互接口和客户端工具。

- Redis命令行客户端 `redis-cli.c` 提供命令行交互界面
- 性能测试工具 `redis-benchmark.c` 进行性能测试和基准测试
- AOF文件检查工具 `redis-check-aof.c` 和 RDB文件检查工具 `redis-check-rdb.c` 进行文件完整性检查
- Redis哨兵模式 `redis-sentinel.c` 提供高可用性监控和故障转移能力
