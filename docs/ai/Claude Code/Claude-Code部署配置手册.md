

# 更新 Homebrew
```bash
brew update
```

# 安装 Node.js
```bash
brew install node
```

# 检查安装是否成功
```bash
node --version
npm --version
```

# 全局安装 Claude Code
```bash
npm install -g @anthropic-ai/claude-code
```

安装完成后，输入以下命令检查是否安装成功：
```bash
claude --version
```
